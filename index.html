FreeCodeCamp Timestamp Micro-Service

Usage

Enter any date as a parameter at the end of this URL, and receive back the unix date and the natural date together in JSON format. For example:

            https://fcc-backend-martensclass.c9users.io/1450137600
                                    OR
            https://fcc-backend-martensclass.c9users.io/December%2015,%202015

Entering either of these will produce the following output:

            { "unix": 1450137600, "natural": "December 15, 2015" }
